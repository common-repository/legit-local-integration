Integrate your WordPress site with Legit Local.

== Description ==

**CONNECT TO YOUR LEGIT LOCAL ACCOUNT**
Trigger Legit Local to prompt customers for feedback when they use any of the configured WordPress platforms

* WooCommerce
* Ninja Forms

**REASONS TO USE THE LEGIT LOCAL PLUGIN**

* Help collect customer feedback
* Increase business
* Improve customer experience

This plugin is perfect for Small Businesses looking to build a strong reputation

== Installation ==
**VIA THE WORDPRESS BACKEND**
1. Download the plugin
2. In the WordPress backend click Plugins > Add New > Upload Plugin > Click 'Choose File' and select the plugin zip file
3. Click 'Install Now'
4. Click 'Activate Plugin'
5. Hover over the Settings in the WordPress backend sidebar and click 'Legit Local'
6. Add your Legit Local API and configure your settings

**VIA FTP**
1. Download the plugin
2. Extract the zip file
3. Upload the `legit-local-plugin` folder to the `/wp-content/plugins/` directory of your website
4. Activate the plugin through in your WordPress backend (WordPress > Plugins > RO Legit Local > Activate
5. Hover over the Settings in the WordPress backend sidebar and click 'Legit Local'
6. Add your Legit Local API and configure your settings
